Global Canopy Height 2024

Published by NASA Jet Propulsion Laboratory

Dataset Overview

This dataset provides global canopy height estimates at 10 m spatial resolution for the year 2024. It is generated using a fusion of GEDI lidar pulses and Copernicus Sentinel-1/2 optical and radar imagery. The model uses a deep learning framework (ResNet-50) trained on over 2 million LiDAR samples globally.

terminal Access with Python

import envibase as eb
ds = eb.open_dataset('nasa/canopy-height-2024')
ds.sel(lat=45.52, lon=-122.67).plot()
