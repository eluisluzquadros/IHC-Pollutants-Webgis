Dúvidas Frequentes

Tudo o que você precisa saber sobre a plataforma

Os dados são exportáveis para outros softwares?

add
Sim, a Envibase permite exportação em diversos formatos padrões do mercado, incluindo GeoJSON, CSV, KML e compatibilidade com AutoCAD e ArcGIS para garantir o fluxo de trabalho colaborativo.

A plataforma funciona offline em campo?

add
Nossa solução híbrida permite a coleta de dados totalmente offline através de nosso app mobile. Assim que houver conexão, os dados são sincronizados automaticamente.

Como garantimos a segurança dos dados?

add
Utilizamos protocolos de segurança de nível bancário e armazenamento em nuvem com redundância geográfica, garantindo que suas informações ambientais estratégicas estejam sempre seguras e disponíveis.
