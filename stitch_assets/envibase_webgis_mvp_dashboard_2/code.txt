Para gerar o código deste MVP da Envibase, utilizaremos uma stack moderna de alta performance para WebGIS: React, Tailwind CSS para a interface, e a biblioteca Mapbox GL JS (ou MapLibre) para o motor do mapa.

Abaixo está a estrutura principal do código em React, focada nos componentes da Sidebar e na integração com o mapa.

1. Estrutura do Componente Principal (Dashboard.jsx)
Este código reflete a separação de escopo, a busca por "Envibases" e o assistente de IA.

2. Aspectos Técnicos do Design no Código:
Sidebar Off-Canvas: Utiliza uma estrutura de aside com flex-col para garantir que o chatbot fique sempre "fixo" na parte inferior.
Tree Filtering: Implementado com um componente recursivo simples que alterna estados de open/close, facilitando a navegação hierárquica por categorias ambientais.
Visual Discovery: Os resultados da busca (Envibases) são renderizados como cards com metadados críticos (Fonte, Data, Tipo de Licença) visíveis antes mesmo do clique.
Integração WebGIS: O mapa usa o estilo dark-v11 do Mapbox para destacar as camadas de dados ambientais que costumam ter cores vibrantes (verde para floresta, azul para água).
3. Próximos Passos para o MVP:
Conexão com API: Integrar o campo de busca com um backend (como FastAPI) que consulta um banco de metadados STAC.
Camada de IA: Utilizar um serviço de OpenAI ou Anthropic para processar os prompts do Assistente e filtrar automaticamente o estado da árvore de categorias.
